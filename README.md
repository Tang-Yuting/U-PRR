# U-PRR
A simple demo for the paper "Learning from Multiple Unlabeled Datasets with Partial Risk Regularization"

## Requirements
* Python 3.7.10
* Numpy 1.20.3
* PyTorch 1.9.0
* Matplotlib 3.3.4

## A Demo for MNIST dataset
You can run an example code of U-PRR on MNIST dataset.

'python experiment.py'
